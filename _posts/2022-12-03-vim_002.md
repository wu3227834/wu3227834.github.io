# <center>vim脚本</center>
```c
vim test.vim
bash test.vim
```
## 1、打印输出和字符串
```c
echo 'Hello world!'
echo 'It''s'
```
vim 可以用 '' 和 "" 来引起一个字符串。两者的区别在于：
- '' 里面不允许有任何转义字符串（'It''s' == "It's"）
- "" 里允许，如 \n 和 \u ...

## 2、变量
变量可以用 let 命令来赋值，如：
```c
let answer=1314
echo 'The meaning of love is' $answer
```

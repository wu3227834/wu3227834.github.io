# <center>jmeter 常用函数</center>

## 获取使用时间戳

```
默认时间戳：
${__time(,)}：精确到毫秒级别，13位
${__time(,)}：精确到秒级别，10位

设置时间格式：
${__time(yyyy-MM-dd,)}
${__time(YMDHMS,)}
```

## 获取线程相关

```
${__BeanShell(ctx.getThread().getThreadName())}：获取线程名称
${__BeanShell(ctx.getThread().getThreadNum())}：获取线程号
```

## 获取属性值

```
${__P(access_token,)}：从属性中取出变量值
```
